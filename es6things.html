一：let and const

The let command is used to declare variables. It is similar with var. But the variables decalared with let is only effective in the code block.

How to use

                {
                    let a = 10;
                    var b = 1;
                }
                
                a // ReferenceError: a is not defined.
                b // 1

The best practice

                for (let i = 0; i < 10; i++) { //
                ...
                }
            
                var a = [];
                for (let i = 0; i < 10;i++) {
                    a[i]=function(){
                        console.log(i);
                    };
                }
                a[6](); // 6
